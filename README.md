YAML 表头设计模板如下：

```
---
atom:
title:
keywords:
  - 
type:
---
```

## YAML 表头解析

### a) atom 

- 前两位是年份
- 后两位是序号

### b) type 

- 批判性思考
	- 读后感
	- 观后感
	- 针对某些概念/观点的深度思考
- 方法论实践：记录方法论的实践感悟
- 方法论总结：针对某一主题/领域的方法论总结

这三者的关系是：

1. 批判性思考形成洞见
2. 实践洞见，形成方法论
3. 方法论在实践过程中会不断迭代优化，形成新的（或者说是更适合我的）方法论
4. 把新的方法论加入到方法论总结中

### c) keywords

列出笔记中提到的核心概念/主题
