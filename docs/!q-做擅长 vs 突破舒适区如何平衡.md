# 锚点句

我现在不确定的是：当 80/20 法则告诉我"做自己擅长的事情"，而《刻意练习》告诉我"在能力边缘工作 / 不断突破舒适区"，**我该如何分配精力**？

这不是文字游戏——两条原则给出的具体行动是**反向的**：

- 80/20：把 80% 精力深耕已擅长的 20% 核心技能
- 刻意练习：必须不断走出舒适区，做"刚好超出当前能力范围"的事

照前者会**路径依赖**——长期靠优势吃饭，竞争力衰退；
照后者会**资源分散**——精力分摊到多个能力区，当下产出降低。

# 张力

**80/20 立场**：
- 个人成就来自杠杆，少数核心技能撬动多数价值
- 试图什么都做反而什么都做不好
- 路径依赖是优势的反面，不是缺陷

**刻意练习立场**：
- 心理表征只在突破舒适区时增长
- 长期停留在舒适区 = 实质性退步（环境在变，你没变）
- 杰出人物的共同点是持续挑战自己

**两者都对，但对象不同**：

- 80/20 在**已经成熟的领域**里讲的是"头部聚焦"
- 刻意练习在**还在成长的领域**里讲的是"边缘扩展"

那么真问题就变成：**如何识别一个领域是"该聚焦"还是"该扩展"？**

# 不确定的地方（最重要）

我不确定的判断标准：

- 一个领域**何时**该停止"突破舒适区"，转入"做擅长"？
  - 标准是市场反馈（已经能赚钱）？
  - 还是主观满意度（已经能流畅工作）？
  - 还是时间长度（10 年是阈值）？
- 反过来，**何时**该从"做擅长"重新进入"突破舒适区"？
  - 危机驱动（被淘汰才动）？
  - 主动驱动（每 N 年强制换一个相邻领域）？
- "相邻能力区" 这个概念**实际可操作吗**？
  - 在我的具体处境里，相邻能力区是什么？怎么识别？
  - 跨度太近 = 没有真挑战；跨度太远 = 无法迁移

# 候选 hypo（DeepSeek 三步走）

DeepSeek 给的解法：**精力 80/20 拆分，时间维度配比**。

> 用 80% 精力深耕已擅长的 20% 核心技能，用 20% 精力探索相邻能力区的「战略支点」。

突破舒适区的三步走：

1. 在新领域识别「**最小关键行动**」（如先掌握核心 20% 知识）
2. 通过刻意练习将其升级为优势技能
3. 纳入个人价值网络，形成**复合杠杆**

特别需要注意的是，要选择**与既有优势协同的领域**：如画家学数字绘画工具——切记不要盲目进入毫无关联的领域。

**这个 hypo 的可疑之处**：

- "80/20 拆分精力" 本身就是用 80/20 解释 80/20——是否同义反复？
- "相邻能力区" 概念依赖直觉判断，没给可操作的识别工具
- 假设了"既有优势" 是稳定的——但优势本身可能正在腐蚀

所以这个 hypo 是**临时压缩的工作模型**，**不代表问题已解决**。
