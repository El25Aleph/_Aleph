## 驱动问题

> **哪些节点暂时没对应到我的认知体系？为什么？**

人性矩阵系列（B 站 up「何夕夕与书」）从基因、神经化学、脑结构、认知、态度、社会规范多层级切入，覆盖广。它不是一个普通的 ref cluster——**它的每个节点天然指向我的某个 moc**（已建或待建）。这张表的作用是把"meta 对应关系"显式化：哪些节点已经在我的认知体系里有锚点，哪些还是悬空的。

悬空的节点 = 下一阶段 cluster 重组或新 moc 的候选。

## 挖矿提示

把 12 篇 ref 整体投喂 AI 时，建议聚焦问题：

> 这 12 个节点的相互依赖关系是什么？哪些是底层（基因 / 神经化学），哪些是涌现层（认知 / 态度 / 道德 / 公共知识）？我的认知体系当前在哪个层级最薄弱？

不要问"它们讲了什么"——那只是复述视频。要问**层级关系与缺口**。

## 维护节奏

- 每读完一本人性 / 神经 / 社会相关的书后，回看这张表：是否能把某个"待挖"节点对应到现有 moc 或新 moc。
- 当某个"待挖"行被填实，意味着一条新 moc 的雏形浮出——这是新建独立 moc 的契机（参照 README §3 的判定标准）。
