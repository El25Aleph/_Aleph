# note

芒格推荐书单：

- 《枪炮、病菌与钢铁》
- 《自私的基因》
- 《冰河世纪》
- 《达尔文的盲点》
- 《影响力》 by 罗伯特·西奥迪尼

这些书的共同点是：

> 研究为什么有些事物能够适应环境，存活下来，甚至在经过很长的时间之后占据统治地位

为了避免「铁锤人」效应：

> 在手里拿着铁锤的人看来，每个问题都像钉子

芒格提出了一个跨学科的「多元思维模型」，用于投资分析，包括：

- 工程学：
  - 冗余备份模型
  - 质量控制理论模型
- 数学：
  - 复利模型
  - 决策树/排列组合模型
- 物理学/化学：临界点、颠覆力矩、自我催化模型
- 生物学：现代达尔文综合模型
- 心理学：
  - 认知误判模型
  - 各种模型共同作用形成的合奏效应
- 会计学：从事商业活动的语言
- 微观经济学
  - 规模优势
  - 社会认同（尤其是在受到压力和困惑时）
- 其它：
  - 逆向思维
  - 二阶思维
  - 能力圈理论

芒格把这套模型做成 checklist, 使用双轨分析法进行分析：

- 理性因素有哪些？
- 潜意识因素有哪些？

## 我的几点思考

- 多元思维模型是“地图 ≠ 疆域”的应用

- 关于能力圈的理解：使用工具的人应该了解工具本身的局限

- 双轨分析是否与《思考，快与慢》中提到的双系统是同一概念？

  - 理性因素：慢思考

  - 潜意识：快思考

- 认同的几点：

  - 应该进行跨学科学习，掌握各个学科一些基础/重要的概念和理论

  - > 现实世界的问题不会恰好落在某个学科的界线之内

  - 两个习惯：

    - 使用逆向思维考虑问题
    - 收集错误判断的例子，从而学会如何在其他人犯错的时候别以他们为榜样

  - > 如果你想要说服别人，要诉诸利益，而非诉诸理性

- 习惯的养成很困难，是因为心理学的「避免不一致性」倾向：由于自由能最小原理，人类的大脑不愿意做出改变

- 在《思考，快与慢》中同样提到了禀赋效应：人往往会自视过高

- 理性思考的原则是第一性原理？
