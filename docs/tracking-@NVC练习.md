# tracking-@NVC练习

> 目的：有意识练习 NVC（观察 / 感受 / 需要 / 请求），发展 Fe。见 [book-@非暴力沟通](book-@非暴力沟通.md)。
>
> **初期特别要盯住的一个摩擦**：我总用「判断」替代「需要」——把"我希望看到的结果和过程"说成对对方的评价（比如「他不懂感激」「他炫耀」）。判断是对人的裁定，需要是我自己缺的东西。练习的核心，就是每次把这层摩擦揪出来。

## 四步提醒（别记混）

- **观察** = 只有事实，不带评价。
- **感受** = 我的情绪。
- **需要** = 我自己缺的、想要的（互惠、被感激、被平等对待……），不是对对方的裁定。
- **请求** = 具体想让对方做什么；有时没有请求，只是对自己设个边界。

## 2026-07-12 · 同事 A（新来的中国同事）

### 小事一：顺路带他上班

- **观察**：我顺路开车带他上班。他上车不说早上好，直接「你先带我去 xxx，我要做 xxx」；下车不说谢谢。（最近有改善。）
- **感受**：不爽、不被尊重。
- **我当时的判断（错误示范）**：他不知感恩、不值得深交。
- **真实需要（提取）**：① 被感激——我主动的善意被看见；② 被平等对待——不被当成理所当然的司机 / 工具。
- **请求**：对他没有。对自己：保持同事边界，工作该配合的照配合，不指望做朋友。

### 小事二：演示翻译软件

- **观察**：我主动给他演示公司翻译软件，正在找中文选项时，他打断——「你给我翻译成英语，我习惯看英语」。
- **感受**：不爽。
- **我当时的判断（错误示范）**：他不懂感激、在炫耀英语。
- **真实需要（提取）**：① 被感激——我主动帮忙，他连谢都没有；② 被平等对待——那句是命令口吻，把我的帮忙当差遣。
- **一个摩擦点**：「炫耀英语」是我替他加的动机，不是事实。他要英语、说习惯英语，是事实；是不是炫耀，我不知道。真正扎我的是命令 + 不致谢的姿态，不是「英语」。红线别挂在「英语」上。

### 整体印象里踩到的价值（不是某一件事，是我的 Fi 值）

- 他 diss 国内公司不懂技术、用盗版软件 → 踩到我「**尊重那些有勇气从零开始摸索的人**」的价值。
- 他爱堆 paper / 专利来显得优秀 → 踩到我「**务实产出 > 应试式炫耀**」的价值。
- 这两条不是 NVC 里的「需要」，是更底层的价值判断，但也是我为什么对他反应大的原因。

### 复盘小结

- 这次的进步：能把事情摊开，让别人帮我把「判断」翻成「需要」。
- 还差的：第一时间我给出的全是判断（不知感恩 / 炫耀），不是需要。这就是初期最该练的地方。
- 边界与提醒：他更多是个「反面教材」，提醒我别成为那样的人。教训提取一次就够了——如果两三年后我还在提这件事，说明我一直没放下。
