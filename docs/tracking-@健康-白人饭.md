# 2025-05-31

地三鲜做法：

1. 土豆和茄子切滚刀，放入微波炉大火叮熟
2. 过油翻炒
3. 彩椒切条
4. 蒜末炝锅
5. 调料：生抽+耗油+味精+糖

# 2026-05-10

蒸茄子的两种做法：

1. 番茄 + 皮蛋
2. 蒜蓉 + 豆瓣酱

注意：

- 蒸茄子时要保证茄子的完整性，不要切开，尤其是茄子头
- 皮蛋和茄子捣碎比单纯把皮蛋切开效果要好
- 同理，茄子蒸熟后不要用刀切开，而是手撕或者用筷子挑开

# 2026-05-04

鸡胸肉 + 西兰花 + 胡萝卜 + 土豆

调料：盐 + 五香粉 + 鸡精

胡萝卜 + 土豆切丁，先过油翻炒一下

蔬菜上面铺鸡胸肉，变色后放西兰花

注意：西兰花选用冷冻好的，最后放是因为它本来是熟的，过早放入会变散；鸡胸肉一定要提前腌制：生抽+耗油+料酒。腌过的肉质会变软，不会柴

# 2026-05-03

鸡蛋打散 + 牛奶 + 吐司

烤箱 200°C 上下风烤 12 min，150°C 再烤 12 min

注意：鸡蛋最好是一个一个的打散，不要一下全放进去，要掌握好吐司和液体的比例；牛奶的作用是去除鸡蛋的腥味

2026-05-31 更新：还可以在中间一片吐司抹上马斯卡彭，做成流心

# 2026-05-02

香蕉两根 + 燕麦 200g + 两个鸡蛋 + 250 ml 牛奶

燕麦泡在牛奶中，打入鸡蛋搅碎，香蕉切片然后碾碎

放在玻璃碗中，碗底和碗边刷上油

烤箱 200°C 上下风烤 30 min，150°C 再烤 20 min

时间不是固定的，最好是隔十几分钟拉出来看一下，水分快干了即可，上表面微微发黄
