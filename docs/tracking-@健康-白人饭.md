# 2026-05-02

香蕉两根 + 燕麦 200g + 两个鸡蛋 + 250 ml 牛奶

燕麦泡在牛奶中，打入鸡蛋搅碎，香蕉切片然后碾碎

放在玻璃碗中，碗底和碗边刷上油

烤箱 200°C 上下风烤 30 min，150°C 再烤 20 min

时间不是固定的，最好是隔十几分钟拉出来看一下，水分快干了即可，上表面微微发黄

# 2026-04-28

胡萝卜 + 西葫芦 + 土豆擦丝

西葫芦放盐杀出水沥干

如果土豆淀粉含量少，可以加一些面粉

热锅 → 放入三丝 → 盘成圆形，中间镂空 → 打入鸡蛋，搅碎，让蛋液渗入蔬菜丝中 → 大概成型后可以用勺子/叉子向下按压，让菜熟的更快 → 调料：盐 + 五香粉 + 黑胡椒
