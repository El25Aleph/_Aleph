# 2026-04-26

搬到 Warstein 之后就一直饱受头皮屑的困扰

之前在国内和 Chemnitz 的时候都没有头皮屑的问题

怀疑和水质有关，Warstein 的水质偏硬

使用了各种洗发水/护发素/头皮保养素，效果不是很好

目前的 lessons learned 有：

- 我的皮肤是干性皮肤，不要用强效去油的洗发水
- 洗完头之后在半湿状态下涂抹头皮保养素，效果比只使用洗发水要好
- 正确洗头：
  - 先在发梢把洗发水搓出泡，再清理头顶
  - 尽量少清理头皮
  - 冲洗头发时要注意把洗发水冲干净，尤其是发际线附近
- 坐飞机时头皮干燥现象尤为严重，可能是因为飞机内的空气比较干燥
