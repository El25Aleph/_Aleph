总结自 B 站 up 主 `小白价值投资` 的系列视频：INTJ 生活指南

鉴于我就是典型的 INTJ 人格，希望总结完这个系列可以对我的生活起到指导意义

INTJ 的 4 个功能分别是：

- 内倾直觉 (NI) → 主导
- 外倾思维 (TE) → 主要辅助
- 内倾情感 (FI) → 次要辅助 
- 外倾感觉 (SE) → 劣势

这四个功能分别代表了什么呢？

- NI → 构建模型
- TE → 执行力和行动力
- FI → 自我审视/价值观
- SE → 环境反馈/感官体验/休息

用车来打比方就好比：

- NI → 地图导航
- TE → 油门/变速箱
- FI → 方向盘
- SE → 地盘和车轮

NI 的特点是预见性与洞察力

NI 就是在头脑中构建一些东西，并将一些点奇妙的串联起来，构建一个模型，然后用这个模型预测未来

TE 决定了 INTJ 的下限

INTJ 常常因为眼高手低而缺乏行动力，一个训练方法就是明确不要一次性做出一个完美的版本，可以先做一个 0.8 版本
